#README
1. To run this program install Interval tree in your python environment
2. Store input file, food db file, exercise db file in same location as your program code
3. Use following command at your command line to run:
	 python compute_blood_sugar.py <input_file> <food_db_file> <exer_db_file>
4. Assumptions : 
	a.This program works only for inputs for a single day, if the date in the input 
	  differs then it throws a custom error
	b. This program will throw error if ID does not exist in the database
	c. Program plots graph at a 30 minute interval
	d. Sample input file : activity.out
